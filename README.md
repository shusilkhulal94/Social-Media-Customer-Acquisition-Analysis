# Social-Media-Customer-Acquisition-Analysis
Analyzed social media marketing performance across TikTok, Instagram and YouTube using SQL, Python and Tableau. Evaluated customer Acquisition metrics, campaign effectiveness, ROAS, CTR, Conversion rate and revenue drivers to identify high performing platforms, camping type and content category.
## Tools
- SQL
- Python
- Tableau

## Key Findings
- TiKTok Generate the higest revenue.
- Paid - ad campaingns produce the highest revenue.
- Entertainment content generate the highest revenue.
- Conversion showed the strongest relationship with revenue.
